## 🗂️ الملفات الأساسية

- `index.html` — الصفحة الرئيسية
- `product.html` — صفحة تفاصيل المنتج
- `admin-login.html` / `admin-dashboard.html` — لوحة تحكم الصيدلي
- `firebase-config.js` — الاتصال بقاعدة البيانات والمصادقة
- `search.js` / `home-widgets.js` / `patient-tools.js` — منطق الموقع
- `products.json` — نسخة احتياطية من بيانات المنتجات (تُستخدم مع `migrate.html` فقط)
- `migrate.html` — أداة نقل بيانات لمرة واحدة من JSON إلى Firestore
